# 3DBotics® TechDojo Curriculum & Lab Gown Promotion System

> **PURPOSE:** To provide the AI Teacher with complete knowledge of the 3DBotics® TechDojo philosophy, monthly curriculum structure, and Lab Gown promotion system.

---

## 🥋 What is TechDojo?

**TechDojo** is a "Technology Dojo" focused on mastery through integrated repetition. It is a special classroom where students learn about **3D printing, AI, and robotics**. The term "Dojo" is a metaphor for a place of learning and discipline, like a martial arts school.

### Core Philosophy
- **Mastery Through Repetition:** Students repeat the same 4-week cycle every month, building deeper understanding each time.
- **Integrated Learning:** Every project combines 3D Modeling, 3D Printing, AI, and Robotics.
- **Discipline & Excellence:** Like a martial arts dojo, students progress through ranks (Lab Gown colors) by demonstrating mastery.

---

## 📅 Monthly Drill Structure

The TechDojo curriculum follows a **continuous monthly cycle**:

### Week 1: 3D Modeling
- **Focus:** Using Tinkercad and other 3D design apps
- **Skills:** Creating digital 3D models, understanding dimensions, designing robot parts
- **Tools:** Tinkercad (web-based), Fusion 360 (advanced)
- **Example Projects:** Design a robot chassis, create custom wheels, model sensor mounts

### Week 2: 3D Printing
- **Focus:** Slicing and hardware operation
- **Skills:** Converting 3D models to printer instructions, operating 3D printers, troubleshooting prints
- **Tools:** Slicing software (Cura, PrusaSlicer), 3D printers
- **Example Projects:** Print the parts designed in Week 1, learn about infill, supports, and print quality

### Week 3: Artificial Intelligence
- **Focus:** Prompt engineering and logic
- **Skills:** Writing effective AI prompts, understanding how AI thinks, using ChatGPT for problem-solving
- **Tools:** ChatGPT, prompt engineering techniques
- **Example Projects:** Write prompts to generate robot code, ask AI to explain electronics concepts, use AI to debug projects

### Week 4: AI-Robotics Integration
- **Focus:** Connecting E-Set components (Arduino, sensors, motors)
- **Skills:** Wiring circuits, programming Arduino, integrating sensors and actuators
- **Tools:** Arduino IDE, breadboards, jumper wires, 3DBotics® E-Set components
- **Example Projects:** Build a line-following robot, create an obstacle-avoiding car, program a robotic arm

## 🤖 What is AI-Robotics?

**AI-Robotics** is the combination of Artificial Intelligence and robotics to create machines that can think, learn, and make decisions. It makes robots smart and adaptable, enabling automation and problem-solving without human intervention.

### Key Idea
Every robot follows this structure:
**Sensor → Microcontroller → Actuator**
(Input → Brain → Output)

**Example:** Distance sensor detects obstacle → Arduino processes data → Motor stops the robot.

---

## 🔧 Basic Robotic Components

### 1. Microcontroller (The Brain)
- **Function:** Processes code and controls everything
- **Examples:** Arduino Nano, Arduino Uno
- **Student Names:** Arduino, the brain, controller

### 2. Sensors (The Eyes and Ears)
- **Function:** Detect the environment (distance, light, sound, etc.)
- **Examples:** 
  - Ultrasonic Sensor (USS) for obstacle avoidance
  - IR Sensor for line following
  - Light sensors for brightness detection
- **Student Names:** USS, ultrasonic, distance sensor, IR sensor

### 3. Motors (The Muscles)
- **Function:** Move parts of the robot (arms, wheels, etc.)
- **Examples:**
  - **TT Gear Motors:** Yellow DC motors with 2 wires (3V-6V)
  - **Servo Motors:** Motors with 3 wires for precise angle control (4.8V-6V)
  - **DC Motors:** General motors for continuous rotation
- **Student Names:** ttmotor, yellow motor, servo, motor

### 4. Motor Driver (The Muscle Controller)
- **Function:** Controls the speed and direction of motors
- **Examples:** L298N Motor Driver
- **Student Names:** L298N, motor driver, driver board

### 5. Power Source (The Energy)
- **Function:** Provides energy to all components
- **Examples:** USB, battery packs, 9V batteries, AA battery holders
- **Student Names:** battery, power, 9V, AA pack

### 6. Breadboard and Jumper Wires (The Connections)
- **Function:** Temporary connections for prototyping circuits
- **Use:** Wire sensors and LEDs without soldering
- **Student Names:** breadboard, wires, jumper wires

### 7. Passive Components (The Helpers)
- **Examples:** Resistors, LEDs, Buzzers
- **Function:** LEDs for feedback, resistors to protect circuits, buzzers for sound
- **Student Names:** LED, light, buzzer, resistor

## 💻 Arduino IDE

The **Arduino Integrated Development Environment (IDE)** is a free, open-source software application that allows you to write, compile, and upload code to Arduino-compatible microcontrollers.

### Key Features
- Write code in C/C++
- Verify (compile) code to check for errors
- Upload code to Arduino boards
- Monitor serial output for debugging

## 📝 What is a Prompt?

A **prompt** is the instruction or input you give to an AI so it knows what to do. It's like asking a question or giving directions to an AI.

### The Golden Rule of Prompts
**The clearer the prompt, the better the AI's answer.**

### Examples
- **Vague Prompt:** "Make me a sandwich."
  - **Problem:** AI doesn't know what kind of bread or filling you want.
- **Clear Prompt:** "Make me a peanut butter and jelly sandwich with whole wheat bread."
  - **Result:** AI can give you a much better answer!

### In Robotics
- **Vague:** "Make a robot."
- **Clear:** "Write Arduino code for a robot car with 2 TT motors controlled by an L298N driver that moves forward when I press a button."
